 🌐 Mi primera página web - CesarGamePixel

¡Hola! Este es mi primer sitio web, creado con HTML básico y publicado con GitHub Pages.

 🚀 ¿Qué contiene este sitio?

Este repositorio incluye una página principal y dos versiones diferentes de mi web. Cada una tiene un estilo distinto para comparar diseños.

 📄 Archivos incluidos:
- `index.html` → Primera versión de mi sitio
- `index V2.0.1.html` → Segunda versión, con mejoras y diseño más moderno
- `index V2.0.2.html` → Tercera versión, con estilo futurista y mejores efectos visuales

 ⚙️ ¿Cómo funciona?

Al visitar el sitio, se abre automáticamente el archivo `index.html`, desde donde puedes acceder a las demás versiones. Solo necesitas un navegador y desacrgar la web como tal

 ℹ️ Información adicional

- No usa bases de datos ni programación avanzada
- Todo el sitio está hecho solo con HTML
